# Agrinho 2026

Site oficial sobre o programa **Agrinho 2026**, que incentiva cidadania, educação e sustentabilidade para crianças e jovens.

## Estrutura
- `index.html` → página principal
- `style.css` → estilos do site
- `script.js` → scripts para interatividade
- `images/` → imagens usadas no site

## Publicação
Você pode publicar este site no GitHub Pages:

1. Crie um repositório chamado `agrinho-2026`.
2. Faça o upload de todos os arquivos.
3. Vá em **Settings → Pages → Source → main branch**.
4. Acesse `https://<seu-usuario>.github.io/agrinho-2026/`.
